Tratamento de Erros

  • exemplo Caso o usuário preencha incorretamente o CEP do destinatário, o fluxo pode retornar um erro na cotação. É importante validar os dados de entrada antes de prosseguir com a cotação.
  • exemplo Se ocorrerem erros na comunicação com ..., o fluxo está configurado para continuar a execução normalmente (propriedade onError definida como continueRegularOutput no nó TOKEN0).
